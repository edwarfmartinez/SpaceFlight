# SpaceFlight

MacOS & iOS app. Get the latest space flight news: international news, the most relevant blogs and the daily report from NASA.

Concepts:

* SwiftUI
* Catalyst Project
* Observable Object
* Web Kit
* Pagination
* XCTest
* Reusable views
